# flare.design

Flare is an AI-native motion canvas for creative work.

We are building a workspace where people and AI agents can compose images, video, text, motion, generation workflows, and reusable creative apps on the same canvas.

## What we are building

- Infinite canvas editor for frames, layers, masks, paths, layout, media, and motion
- AI image and video generation with reusable Recipes
- Timeline-based motion authoring and cloud MP4 rendering
- Workspace assets, team collaboration, permissions, and billing
- Remote MCP access so agents can read project context, place media, edit canvases, and start controlled generation or render jobs

## For creators

Use Flare to explore ideas, generate visual assets, arrange them on a canvas, add motion, and export finished creative work.

[Open Flare](https://app.flare.design)

## For agents and developers

Flare exposes project, canvas, asset, generation, motion, and render capabilities through an MCP endpoint.

[Read the MCP guide](https://flare.design/docs/configure-flare-design-mcp)
